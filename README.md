# 🚪 Smart Door Automation System (Arduino C++)

## 📌 Overview
This project implements an automated door system using Arduino. The system detects motion using a sensor and controls a servo motor to open and close the door automatically.

It is designed to improve convenience, security, and reduce manual effort.

---

## 🎯 Objectives
- Automate door opening and closing
- Detect human presence using sensors
- Implement real-time control using Arduino
- Improve system reliability with optimized logic

---

## 🛠️ Components Used
- Arduino Uno
- PIR / IR Sensor
- Servo Motor (SG90)
- Buzzer (optional)
- Jumper Wires
- Breadboard

---

## ⚙️ Working Principle
1. The sensor detects motion near the door.
2. Arduino processes the input signal.
3. If motion is detected:
   - Servo motor rotates → Door opens
   - Buzzer activates (optional)
4. After a delay:
   - Door closes automatically
